This Read me contains a brief description of the project, as well as instruction on how to launch and play.

OVERVIEW:
    This is a web-based curling simulation game built with HTML5 Canvas and JavaScript. 
    Players can shoot curling stones using mouse controls, with real-time collision detection and physics-based movement simulation. 
    The game features a split-screen view showing both a shooting area and a zoomed close-up view of the target rings.
    It supports multiplayer gameplay through Socket.IO, allowing players to join as home team, visitor team, or spectators.

Technologies used:
     Node.js, Socket.IO for real-time multiplayer communication, HTML5 Canvas for rendering, 
    and vanilla JavaScript (ES6) for game logic and physics simulation. This is all published in github

To launch:
    Run this command in terminal/command prompt to install dependencies:
    --> npm install

    Then run this command and follow the link:
    --> npm start 

    Once this is done, you will see a link, follow it. 

Game instructions:
    Game instructions:
    Join as HOME and VISITOR, alternate between HOME and VISITOR while taking turns throwing the stone.
    Click and drag with your mouse to aim and set the power of your throw.
